# <img src="https://raw.githubusercontent.com/Anto426/Anto426/main/asset/icon2.gif" width="70px" /> Important Notes;

```sh
root@anto426: ~/auto-setup-lt (main⚡)$ cat rules.txt

- 🔤 Make sure you are using a standard Nerd Font in your terminal.
- 💾 Always create a system backup before running terminal bootstrap scripts.
- 🚫 Uninstallation tasks are ignored to prevent damage to default system tools.
- 🖼️ The script places '~/neofetch-random.sh' to render fastfetch logos.
```

# <img src="https://raw.githubusercontent.com/Anto426/Anto426/main/asset/icon3.gif" width="70px" /> Installation;

```sh
root@anto426: ~/auto-setup-lt (main⚡)$ ./bootstrap.sh
```

### 🏔️ Arch-based Distributions
*Arch Linux, EndeavourOS, Manjaro, etc.*

```bash
sudo pacman -Syu --noconfirm
bash -c "$(curl -fSL https://raw.githubusercontent.com/Anto426/auto-setup-LT/main/arch.sh)"
```

### 🍥 Debian-based Distributions
*Ubuntu, Kali Linux, Linux Mint, etc.*

```bash
sudo apt update && sudo apt upgrade -y
bash -c "$(curl -fSL https://raw.githubusercontent.com/Anto426/auto-setup-LT/main/debian.sh)"
```
